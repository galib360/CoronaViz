About CoronaVIZ

Data Source link for Bangladesh data


CoronaVIZ is a Rongmistry Interactive toy project. CoronaVIZ interactively visualizes geospatial data along with the confirmed cases, deaths and recovered cases of the current COVID-19 outbreak in a global scale as well as in a local scale (in the country of Bangladesh with district wise data). Data is static and gathered from the links provided. There is a script written in python in order to automatically update the interactive dashboard with latest data every morning. The visualization is achieved with D3.js.
